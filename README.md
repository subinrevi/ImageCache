
# ImageLoader

The goal of this framework is to understand how image caching works in iOS including low level design, implementation and high level design

## Table of Contents

## Features

*   Local caching using NSCache
*   Permanent caching in disk storage using FileManager
*   Option to download and save images from url/url's
